1. Kalimat dengan 是 (shì)

Subjek + 是 + Objek

Kata 是 digunakan untuk menyatakan "adalah". Pola ini paling dasar dalam Bahasa Mandarin.

我是学生。
Saya adalah murid.

2. Bertanya dengan 吗 (ma)

Kalimat pernyataan + 吗？

Tambahkan 吗 di akhir kalimat pernyataan untuk membuat pertanyaan ya/tidak.

Bagian A — Soal 1
我今天很高兴。

Apa arti kalimat di atas?

A. Saya hari ini sangat lelah.
B. Saya hari ini sangat senang.
C. Saya hari ini sangat sibuk.
D. Saya hari ini sangat lapar.
Bagian C — Susun Kalimat

Susun kata-kata berikut menjadi kalimat yang benar:

喜欢 我 中国菜 吃
